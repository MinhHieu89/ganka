## Decisions Made
- **User dropdown in both header and footer:** SiteHeader has a compact user dropdown (avatar + name + logout), NavUser in SidebarFooter provides the full dashboard-01 experience. Both read from the same authStore.
- **IconSelector instead of IconChevronsUpDown:** @tabler/icons-react does not have IconChevronsUpDown; IconSelector is the equivalent up/down chevron icon.
- **Route-based breadcrumbs:** Used `useRouterState().location.pathname` split into segments with a `segmentToI18nKey` lookup table for i18n translation. Layout route segments (like `_authenticated`) are filtered out.
- **Disabled items with Tooltip:** Disabled nav items use a Tooltip wrapper with "Coming soon" text and opacity-50 + cursor-not-allowed styling on the SidebarMenuButton.
