API Endpoints:
```
POST /api/clinical/{visitId}/images  -- IFormFile upload with [FromForm] imageType, eyeTag
GET  /api/clinical/{visitId}/images  -- returns MedicalImageDto[] with SAS URLs
DELETE /api/clinical/images/{imageId}
GET  /api/clinical/image-comparison?patientId=&visitId1=&visitId2=&imageType=
GET  /api/public/osdi/{token}  -- returns OsdiQuestionnaireDto (no auth)
POST /api/public/osdi/{token}  -- submits answers (no auth)
```

MedicalImageDto shape:
```typescript
interface MedicalImageDto {
  id: string
  visitId: string
  type: number  // ImageType enum
  eyeTag: number | null  // EyeTag enum
  originalFileName: string
  url: string  // SAS URL
  contentType: string
  fileSize: number
  description: string | null
  createdAt: string
}
```
