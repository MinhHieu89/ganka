**Route group structure:**

```csharp
public static IEndpointRouteBuilder MapAuthApiEndpoints(this IEndpointRouteBuilder app)
{
    var authGroup = app.MapGroup("/api/auth");
    var adminGroup = app.MapGroup("/api/admin").RequireAuthorization();

    MapAuthFlowEndpoints(authGroup);
    MapAdminUserEndpoints(adminGroup);
    MapAdminRoleEndpoints(adminGroup);
    MapAdminPermissionEndpoints(adminGroup);

    return app;
}
```

**Auth flow endpoints refactored pattern:**

Each endpoint that returns `Result` or `Result<T>` uses `.ToHttpResult()` or `.ToHttpResult<T>()`.

- **Login** (`/login` on authGroup, POST, no auth required):
  ```csharp
  authGroup.MapPost("/login", async (LoginCommand command, IMessageBus bus, HttpContext ctx) =>
  {
      var enriched = command with { IpAddress = ctx.Connection.RemoteIpAddress?.ToString() };
      var result = await bus.InvokeAsync<Result<LoginResponse>>(enriched);
      return result.ToHttpResult();
  });
  ```

- **Logout** (`/logout` on authGroup, POST, auth required):
  ```csharp
  authGroup.MapPost("/logout", async (IMessageBus bus, HttpContext ctx) =>
  {
      if (!ctx.TryGetUserId(out var userId))
          return Results.Unauthorized();
      var result = await bus.InvokeAsync<Result>(new LogoutCommand(userId));
      return result.ToHttpResult();
  }).RequireAuthorization();
  ```

- **Refresh** (`/refresh` on authGroup, POST, no auth required):
  ```csharp
  authGroup.MapPost("/refresh", async (RefreshTokenCommand command, IMessageBus bus) =>
  {
      var result = await bus.InvokeAsync<Result<RefreshTokenResponse>>(command);
      return result.ToHttpResult();
  });
  ```

- **GetCurrentUser** (`/me` on authGroup, GET, auth required):
  ```csharp
  authGroup.MapGet("/me", async (IMessageBus bus, HttpContext ctx) =>
  {
      if (!ctx.TryGetUserId(out var userId))
          return Results.Unauthorized();
      var result = await bus.InvokeAsync<Result<UserDto>>(new GetCurrentUserQuery(userId));
      return result.ToHttpResult();
  }).RequireAuthorization();
  ```

- **UpdateLanguage** (`/language` on authGroup, PUT, auth required):
  ```csharp
  authGroup.MapPut("/language", async (UpdateLanguageCommand command, IMessageBus bus, HttpContext ctx) =>
  {
      if (!ctx.TryGetUserId(out var userId))
          return Results.Unauthorized();
      var enriched = command with { UserId = userId };
      var result = await bus.InvokeAsync<Result>(enriched);
      return result.ToHttpResult();
  }).RequireAuthorization();
  ```

**Route group structure:**
```csharp
public static IEndpointRouteBuilder MapAuditApiEndpoints(this IEndpointRouteBuilder app)
{
    var adminGroup = app.MapGroup("/api/admin").RequireAuthorization();

    // Audit logs: query with filtering and cursor-based pagination
    adminGroup.MapGet("/audit-logs", async ([AsParameters] GetAuditLogsQuery query, IMessageBus bus) =>
    {
        var result = await bus.InvokeAsync<GetAuditLogsResponse>(query);
        return Results.Ok(result);
    });

    // Audit logs: export as CSV file download
    adminGroup.MapGet("/audit-logs/export", async ([AsParameters] ExportAuditLogsQuery query, IMessageBus bus) =>
    {
        var result = await bus.InvokeAsync<ExportAuditLogsResponse>(query);
        return Results.File(result.FileContents, "text/csv", result.FileName);
    });

    // Access logs: query with filtering and cursor-based pagination
    adminGroup.MapGet("/access-logs", async ([AsParameters] GetAccessLogsQuery query, IMessageBus bus) =>
    {
        var result = await bus.InvokeAsync<GetAccessLogsResponse>(query);
        return Results.Ok(result);
    });

    return app;
}
```

**Notes:**
- Audit endpoints return response objects directly (not Result<T>), so they don't need ResultExtensions. They simply return `Results.Ok(result)` or `Results.File(...)`.
- The export endpoint uses `Results.File()` which is not a Result pattern -- keep as-is.
- Route preservation: Old `/api/admin/audit-logs` → New: `adminGroup.MapGet("/audit-logs", ...)` where adminGroup = `app.MapGroup("/api/admin")`. Final route: `/api/admin/audit-logs` -- SAME.
- Add `using Shared.Presentation;` even though Audit doesn't currently need ResultExtensions, to be consistent and ready for future extensions.
