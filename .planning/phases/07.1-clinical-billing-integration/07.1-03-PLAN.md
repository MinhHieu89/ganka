  <behavior>
    HandleVisitCreated:
    - VisitCreatedIntegrationEvent -> creates Draft invoice via Invoice.Create() with visit info
    - Looks up "CONSULTATION" from IServiceCatalogRepository
    - If consultation service found: adds line item (Department.Medical, sourceId=VisitId, sourceType="Visit")
    - If consultation service not found: creates invoice without line item (edge case)
    - Uses IInvoiceRepository.GetNextInvoiceNumberAsync for invoice number
    - Uses ICurrentUser.BranchId for BranchId

    HandleVisitCancelled:
    - VisitCancelledIntegrationEvent -> finds invoice by VisitId
    - If invoice exists and is Draft: calls invoice.Void()
    - If invoice not found: no-op (idempotent)
    - If invoice already Voided: no-op (idempotent)

    HandleDrugDispensed:
    - DrugDispensedIntegrationEvent -> finds or creates invoice by VisitId
    - Adds one line item per drug in Items list (Department.Pharmacy, sourceId=VisitId, sourceType="Dispensing")
    - Uses DrugName/DrugNameVi for description/descriptionVi, UnitPrice and Quantity from event

    HandleOtcSaleCompleted:
    - OtcSaleCompletedIntegrationEvent -> creates standalone invoice with VisitId=null
    - PatientId from event (nullable for anonymous), CustomerName as PatientName
    - Adds one line item per drug (Department.Pharmacy, sourceId=OtcSaleId, sourceType="OtcSale")

    HandleGlassesOrderCreated:
    - GlassesOrderCreatedIntegrationEvent -> finds or creates invoice by VisitId
    - Adds one line item per item in Items list (Department.Optical, sourceId=OrderId, sourceType="GlassesOrder")

    HandleTreatmentSessionCompleted:
    - TreatmentSessionCompletedIntegrationEvent -> uses @event.VisitId and @event.SessionFeeAmount directly (extended in Plan 01b)
    - If VisitId is null: log warning and skip (treatment without visit cannot be billed to a visit invoice)
    - If VisitId has value: find or create invoice by VisitId using get-or-create pattern
    - Adds line item with SessionFeeAmount as unitPrice (Department.Treatment, sourceId=SessionId, sourceType="TreatmentSession")
    - Description: treatment type name, DescriptionVi: Vietnamese equivalent
  </behavior>
