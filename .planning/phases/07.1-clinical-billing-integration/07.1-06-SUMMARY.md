## Decisions Made
- Used Checkbox for show-inactive toggle (consistent with other table filters), Switch for the active/inactive form control
- SignalR hook invalidates TanStack Query caches on events (InvoiceCreated, LineItemAdded, InvoiceVoided) rather than managing state directly
- Connection status uses Badge with outline variant and colored dot for minimal visual footprint
- Re-joins cashier-dashboard group on reconnect since SignalR group membership is lost when connection drops
