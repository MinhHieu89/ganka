   - Run `npm create @tanstack/start@latest frontend` (select SPA mode)

2. Configure `vite.config.ts`:
   - Use `@tanstack/start-vite` plugin with `spa: { enabled: true }`
   - Add `@tailwindcss/vite` plugin
   - Set path alias: `@` -> `/src`

3. Install all dependencies:
   ```
   npm install @tanstack/react-query @tanstack/react-table zustand
   npm install react-hook-form @hookform/resolvers zod
   npm install i18next react-i18next i18next-http-backend i18next-browser-languagedetector
   npm install openapi-fetch sonner
   npm install -D openapi-typescript @tailwindcss/vite
   ```

4. Initialize shadcn/ui per user decision (Maia style, Stone base color, Green theme, no border radius):
   - Run `npx shadcn@latest init` -- configure for Vite + TanStack Start
   - Set `components.json` with:
     - Style: "default" (Maia not available in CLI -- apply Stone base + Green theme via CSS variables)
     - tailwindCSS path: `src/styles/globals.css`
     - Components alias: `@/shared/components/ui`
     - Utils alias: `@/shared/lib/utils`
     - No border radius: set `--radius: 0` in CSS variables
   - Add essential shadcn components: button, input, label, card, sidebar, dropdown-menu, dialog, table, badge, separator, sheet, skeleton, tooltip, checkbox, select, sonner
   - Use Context7 for shadcn/ui Vite installation guide

5. Create `src/shared/lib/api-client.ts`:
   - Create openapi-fetch client pointing to `import.meta.env.VITE_API_URL` (default: `http://localhost:5000`)
   - Export typed fetch client for use with TanStack Query

6. Create `src/shared/stores/authStore.ts`:
   - Zustand store with: user (null | { id, email, fullName, permissions, preferredLanguage }), accessToken (string | null), isAuthenticated (boolean), setAuth/clearAuth actions
   - Persist nothing to localStorage for security (access token is in-memory only)

7. Create `src/shared/stores/sidebarStore.ts`:
   - Zustand store with: isCollapsed (boolean), toggle action
   - Persist collapse state to localStorage

8. Configure `globals.css` with Tailwind v4 directives, Stone base color with Green accent, `--radius: 0` (no border radius per user decision). Use Inter font.

Use Context7 for TanStack Start SPA mode, shadcn/ui Vite installation, and Zustand store patterns.
