  <files>backend/src/Modules/Treatment/Treatment.Domain/Entities/TreatmentPackage.cs</files>
  <action>
    Create TreatmentPackage.cs following the GlassesOrder/Invoice aggregate pattern:

    - namespace Treatment.Domain.Entities
    - Inherits AggregateRoot, implements IAuditable
    - Private backing fields:
      - _sessions (List of TreatmentSession)
      - _versions (List of ProtocolVersion) -- exposed as IReadOnlyList
      - _cancellationRequest (CancellationRequest?) -- nullable, one per package
    - Properties (all private set):
      - ProtocolTemplateId (Guid) -- FK to TreatmentProtocol
      - PatientId (Guid) -- FK to Patient module
      - PatientName (string) -- denormalized for display
      - TreatmentType (TreatmentType enum)
      - Status (PackageStatus enum, default Active)
      - TotalSessions (int, 1-6)
      - PricingMode (PricingMode enum)
      - PackagePrice (decimal)
      - SessionPrice (decimal)
      - MinIntervalDays (int)
      - ParametersJson (string) -- default structured params from template
      - VisitId (Guid?) -- optional link to originating visit
      - CreatedById (Guid) -- doctor who created
      - IAuditable: CreatedAt, UpdatedAt, CreatedById, UpdatedById
      - BranchId
    - Computed properties (read-only, NOT stored in DB):
      - Sessions (IReadOnlyList from _sessions)
      - Versions (IReadOnlyList from _versions)
      - CancellationRequest (from _cancellationRequest)
      - SessionsCompleted => _sessions.Count(s => s.Status == SessionStatus.Completed)
      - SessionsRemaining => TotalSessions - SessionsCompleted
      - IsComplete => SessionsCompleted >= TotalSessions

    - Private parameterless constructor for EF Core
    - Static Create factory method with all required params
    - RecordSession method:
      - Calls EnsureActive()
      - Creates TreatmentSession via factory, adds to _sessions
      - If IsComplete after adding, sets Status = Completed, raises TreatmentPackageCompletedEvent
      - Always raises TreatmentSessionCompletedEvent with consumable list
      - Returns the created session
    - Modify method (for TRT-07):
      - Calls EnsureModifiable()
      - Creates ProtocolVersion snapshot of current state, adds to _versions
      - Applies changes (totalSessions, parametersJson, etc.)
      - SetUpdatedAt()
    - Pause/Resume methods: transition Active<->Paused
    - RequestCancellation method: sets Status to PendingCancellation, creates CancellationRequest
    - ApproveCancellation method: sets Status to Cancelled
    - RejectCancellation method: sets Status back to Active
    - MarkAsSwitched method: sets Status to Switched
    - CalculateRefundAmount(decimal deductionPercent) method:
      - If PricingMode.PerSession: (SessionsRemaining * SessionPrice) * (1 - deductionPercent/100)
      - If PricingMode.PerPackage: (PackagePrice * SessionsRemaining / TotalSessions) * (1 - deductionPercent/100)
    - EnsureActive/EnsureModifiable guard methods (throw InvalidOperationException)
      - EnsureModifiable allows Active and Paused states
      - EnsureActive only allows Active state
  </action>
