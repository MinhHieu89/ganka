**1. Create a new hook `useOsdiTokenHub` in use-osdi-hub.ts:**

Add alongside the existing `useOsdiHub` hook (do NOT modify the existing hook):

```typescript
export function useOsdiTokenHub(
  token: string | undefined,
  onScoreReceived: (score: number) => void
) {
  useEffect(() => {
    if (!token) return;

    const connection = new HubConnectionBuilder()
      .withUrl("/hubs/osdi")
      .withAutomaticReconnect()
      .build();

    let mounted = true;

    connection.on("OsdiTokenSubmitted", (data: { token: string; score: number; severity: string }) => {
      if (mounted) {
        onScoreReceived(data.score);
      }
    });

    connection.start()
      .then(() => connection.invoke("JoinToken", token))
      .catch(console.error);

    return () => {
      mounted = false;
      connection.invoke("LeaveToken", token).catch(() => {});
      connection.stop().catch(() => {});
    };
  }, [token]); // Note: onScoreReceived should be stable (useCallback in parent)
}
```

Check how the existing `useOsdiHub` hook creates its SignalR connection -- follow the same pattern (same HubConnectionBuilder config, same URL, same import from @microsoft/signalr). The hub URL is mapped in the backend -- check Program.cs or wherever `MapHub<OsdiHub>` is configured to get the exact path.
