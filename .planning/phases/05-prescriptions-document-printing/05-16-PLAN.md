  - OS row: same fields

  - Far PD (Input decimal)
  - Near PD (Input decimal)
- Lens type (Select): SingleVision, Bifocal, Progressive, Reading
- Notes (Textarea)

- On mount (or via a "Tu dong dien tu do refraction" / "Auto-fill from refraction" button):
  - Find manifest refractions (refractionType === 0) for OD (eyeTag === 0) and OS (eyeTag === 1)
  - Pre-fill form: odSph from OD manifest sph, odCyl from OD cyl, odAxis from OD axis, odAdd from OD add
  - Pre-fill osSph from OS manifest sph, etc.
  - Pre-fill farPd from OD manifest pd (or average if both eyes have pd)
- Doctor can then modify values before saving

Layout: OD/OS side-by-side grid (reference RefractionForm for styling pattern).
