**pharmacy-api.ts**: TanStack Query hooks for drug catalog:
- `pharmacyKeys` factory: `{ all: ['pharmacy'], drugs: ['pharmacy', 'drugs'], drugSearch: (term) => ['pharmacy', 'drugs', 'search', term] }`
- `useDrugCatalogList()`: useQuery for GET /api/pharmacy/drugs (all active drugs)
- `useDrugCatalogSearch(term: string)`: useQuery for GET /api/pharmacy/drugs/search?term=xxx, enabled only when term.length >= 2, debounced
- `useCreateDrugCatalogItem()`: useMutation for POST /api/pharmacy/drugs, invalidates pharmacyKeys.drugs on success
- `useUpdateDrugCatalogItem()`: useMutation for PUT /api/pharmacy/drugs/{id}, invalidates pharmacyKeys.drugs on success

Use openapi-fetch api client (same pattern as clinical-api.ts). Include onError toast.error callback on all mutations per established pattern.
