<decisions>
## Implementation Decisions

### Session Persistence
- Session is lost on every page refresh — user is always redirected to login
- Fix must ensure session persists across page refreshes (AUTH-04 compliance)
- Sessions always persist until the 30-min inactivity timeout
- Add "Remember me" checkbox on login form — allows user to stay logged in longer
- "Remember me" duration should follow project requirements (refresh token based persistence)

### shadcn/ui Visual Alignment
- All pages feel off from shadcn/ui design language — not specific pages
- Issues span: spacing/padding, colors/contrast, component styling, and overall layout
- **Reset to shadcn/ui default theme** — drop the custom Stone base + Green accent theme
- **Switch to rounded corners** — drop zero border radius, use shadcn/ui default border radius
- Reference https://ui.shadcn.com/create as the visual benchmark for all pages
- Every page should be audited and fixed: login, dashboard/sidebar, patient list, patient profile, scheduling/calendar, public booking

### Claude's Discretion
- Exact CSS variable values for the default shadcn/ui theme
- Which components need restyling vs which are already correct
- Order of pages to fix
- Whether to fix theme globally (CSS variables) or per-component

</decisions>
