       - `hsl(var(--border))` -> `var(--border)`
       - `hsl(var(--background))` -> `var(--background)`
       - `hsl(var(--foreground))` -> `var(--foreground)`
       - `hsl(var(--primary))` -> `var(--primary)`
       - `hsl(var(--muted))` -> `var(--muted)`
       - etc.
    2. Remove all hardcoded `border-radius: 0` or `borderRadius: 0` in FullCalendar CSS/styles. Let components inherit the theme's rounded corners from `--radius`. If FullCalendar events or buttons need specific radius, use `calc(var(--radius) - 2px)` or similar.
    3. Replace any `bg-stone-*`, `text-stone-*`, `border-stone-*` classes with semantic equivalents:
       - `bg-stone-50` -> `bg-muted` or `bg-background`
       - `bg-white` -> `bg-card` or `bg-background`
       - `text-stone-600` -> `text-muted-foreground`
