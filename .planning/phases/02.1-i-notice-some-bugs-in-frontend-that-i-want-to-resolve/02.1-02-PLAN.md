**AppointmentCalendar.tsx -- Fix hsl() wrapping and hardcoded border-radius:**
1. Replace ALL `hsl(var(--...))` with `var(--...)` in the inline style block. The CSS variables store oklch values that should NOT be wrapped in hsl(). Specific replacements:
   - `hsl(var(--border))` -> `var(--border)`
   - `hsl(var(--accent) / 0.15)` -> `color-mix(in oklch, var(--accent) 15%, transparent)` (or use `oklch(from var(--accent) l c h / 0.15)` if the browser supports it -- safer to use `color-mix`)

   - `hsl(var(--muted) / 0.5)` -> `color-mix(in oklch, var(--muted) 50%, transparent)`

   - `hsl(var(--secondary-foreground))` -> `var(--secondary-foreground)`
   - `hsl(var(--primary-foreground))` -> `var(--primary-foreground)`
   - `hsl(var(--muted-foreground))` -> `var(--muted-foreground)`
