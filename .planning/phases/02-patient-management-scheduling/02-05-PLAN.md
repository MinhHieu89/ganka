         - `initialView="timeGridWeek"` with headerToolbar for prev/next/today and week/day toggle

         - `eventOverlap={false}`, `selectOverlap={false}`

       - `select` handler: opens AppointmentBookingDialog with pre-filled start/end time

       - `eventDrop` handler: calls useRescheduleAppointment with new start time, shows error toast if double-booking
       - Style non-business hours with gray background

    5. Create `AppointmentBookingDialog.tsx`:
       - Dialog form for booking an appointment
       - Fields:
         - Patient: Command/Popover autocomplete search (reuse usePatientSearch from patient module)
         - Doctor: pre-filled from calendar's selected doctor (editable via DoctorSelector)
         - Appointment Type: Select dropdown from useAppointmentTypes (shows Vietnamese name, duration in parentheses)

         - Notes: optional textarea

       - Zod validation: patient required, doctor required, start time in future
       - On submit: useBookAppointment mutation
       - Error handling: show toast for double-booking (409 Conflict) or outside hours (400 Validation)
       - On success: close dialog, refetch calendar events

    6. Create `AppointmentDetailDialog.tsx`:

         - "Complete" button (if status is Confirmed)
       - Status badge with color coding
