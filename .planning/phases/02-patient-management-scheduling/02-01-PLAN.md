Return `Result<Guid>`.

       **Note on patient code generation:** In the handler, after first SaveChangesAsync: query `GetMaxSequenceNumberForYearAsync(currentYear)`, compute next = max + 1 (or 1 if null), set `patient.Year = currentYear`, `patient.SequenceNumber = next`, call `patient.SetPatientCode(currentYear)`, then second SaveChangesAsync. The unique constraint on (Year, SequenceNumber) is the race condition safety net.

       **UpdatePatient.cs:** Command + Validator + Handler. Load patient with tracking, call patient.Update(), save. Return `Result`.
