  <action>
    Create each handler following existing CQRS pattern (command record + handler class).

    SkipRefraction: PUT /api/clinical/{visitId}/skip-refraction. Body: { reason: int, freeTextNote?: string }. Calls visit.SkipRefraction then visit.AdvanceStage(DoctorExam).

    UndoRefractionSkip: PUT /api/clinical/{visitId}/undo-refraction-skip. No body. Calls visit.UndoRefractionSkip().

    RequestImaging: POST /api/clinical/{visitId}/request-imaging. Body: { note?: string, services: string[] }. Calls visit.RequestImaging then visit.AdvanceStage(Imaging).

    CompleteImagingServices: PUT /api/clinical/{visitId}/complete-imaging. No required body. Calls visit.AdvanceStage(DoctorReviewsResults).

    ConfirmVisitPayment: POST /api/clinical/{visitId}/confirm-payment. Body: { amount: decimal, paymentMethod: int, amountReceived: decimal, splitGlasses: bool }. Creates VisitPayment. Checks if visit has drug prescriptions and/or optical prescriptions. Calls visit.ActivatePostPaymentTracks(hasDrugs, hasGlasses). If neither track active, advances to Done.

    DispensePharmacy: PUT /api/clinical/{visitId}/dispense-pharmacy. Body: { dispensedItems: [{drugName, quantity, isDispensed}], note?: string }. Creates PharmacyDispensing. Calls visit.CompleteDrugTrack(). If visit.IsComplete, advances to Done.

    ConfirmOpticalOrder: POST /api/clinical/{visitId}/confirm-optical-order. Body: { lensType, frameCode, lensCost, frameCost, totalPrice }. Creates OpticalOrder. Advances glasses track to CashierGlasses.

    ConfirmGlassesPayment: POST /api/clinical/{visitId}/confirm-glasses-payment. Body: { amount, paymentMethod, amountReceived }. Creates VisitPayment (type=Glasses). Advances to OpticalLab stage.

    CompleteOpticalLab: PUT /api/clinical/{visitId}/complete-optical-lab. Body: { qualityChecklist: string[] }. Advances to ReturnGlasses stage.

    CompleteHandoff: PUT /api/clinical/{visitId}/complete-handoff. Body: { prescriptionVerified, frameCorrect, patientConfirmedFit }. Creates HandoffChecklist. Calls visit.CompleteGlassesTrack(). If visit.IsComplete, advances to Done.

    Update SignOffVisit to auto-advance from Prescription to Cashier (was Rx -> next stage).
