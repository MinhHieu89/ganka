    **Stage3DoctorExamView.tsx:**
    Uses StageDetailShell wrapper. Props: visit data.

    **Refraction summary card (top of view):**
    - If Stage 2 completed (visit.refractionSkipped === false): read-only two-column card showing OD/OS with SPH, CYL, AXIS, uncorrected VA, corrected VA. Use existing RefractionSection in disabled/readonly mode.
    - If Stage 2 skipped (visit.refractionSkipped === true): compact amber inline notice: `"Đo khúc xạ đã bỏ qua — Lý do: [reason] · [actor] · [time]"`. Use `bg-amber-50 border-amber-300` styling.

    **ICD-10 diagnosis section:**
    - Reuse existing Icd10Combobox component for search typeahead
    - Selected diagnoses as pill tags below input. Each tag has `x` to remove.
    - Tags are teal (`bg-teal-100 text-teal-800`) by default
    - At least one tag required to enable `Kê đơn ›` button
    - Search input remains visible after selections

    **Examination notes:**
    - Free-text textarea, optional, no character limit
    - Does not block forward button

    **Validation message:**
    - Error (no ICD-10): `"Chẩn đoán bắt buộc — chọn ít nhất một mã ICD-10 để tiếp tục."`
    - While searching: `"Đang tìm kiếm — chọn mã từ danh sách để tiếp tục."`
    - Success: `"Sẵn sàng — chuyển kê đơn, hoặc yêu cầu CĐHA nếu cần."`

    **Bottom bar:**
    1. `Lưu nháp` — always enabled, saves notes + diagnoses
    2. `Chuyển CĐHA ›` — secondary outlined button, ALWAYS enabled regardless of ICD-10. Calls useRequestImaging. Opens a small modal asking for optional referral note and service checkboxes (OCT, IOP, etc.).
    3. `Kê đơn ›` — primary black button, disabled until >= 1 ICD-10 tag. Calls useAdvanceStage from DoctorExam(2) to Prescription(5). BUT: if visit.imagingRequested === true, this button should be hidden (doctor must go through 4a->4b first).

    **Branch decision logic:**
    - If imagingRequested is false: show both buttons. `Kê đơn ›` goes to Prescription(5). `Chuyển CĐHA ›` goes to Imaging(3).
    - If imagingRequested is true and visit is back at DoctorExam (shouldn't happen per flow, but guard): only show `Chuyển CĐHA ›`.

    Update the route file to render Stage3DoctorExamView for currentStage === 2.
