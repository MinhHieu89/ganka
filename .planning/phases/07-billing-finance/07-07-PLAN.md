    1. **InvoiceRepository.cs**: Implements IInvoiceRepository:
       - Constructor injects BillingDbContext
       - GetByIdAsync: Include(LineItems).Include(Payments).Include(Discounts) + also include Refunds if tracked. Use FirstOrDefaultAsync(i => i.Id == id)
       - GetByVisitIdAsync: Same includes, filter by VisitId
       - GetByInvoiceNumberAsync: Same includes, filter by InvoiceNumber
       - GetByShiftIdAsync: Filter by CashierShiftId, include line items
       - GetPendingByPatientIdAsync: Filter by PatientId AND Status == Draft
       - GetNextInvoiceNumberAsync: Query MAX InvoiceNumber for given year, parse sequence, increment. Format: HD-{year}-{seq:D5}. If no invoices for the year, start at HD-YYYY-00001.
       - Add: _context.Invoices.Add(invoice)
       - Update: _context.Invoices.Update(invoice)

    2. **PaymentRepository.cs**: Implements IPaymentRepository:
       - Constructor injects BillingDbContext
       - GetByIdAsync: FirstOrDefaultAsync by Id
       - GetByInvoiceIdAsync: Where InvoiceId, ToListAsync
       - GetByShiftIdAsync: Where CashierShiftId, ToListAsync
       - GetByTreatmentPackageIdAsync: Where TreatmentPackageId (not null), ToListAsync
       - Add: _context.Set of Payment.Add(payment)
       - Update: _context.Set of Payment.Update(payment)

    Follow the established repository pattern from PharmacyDbContext repositories.
