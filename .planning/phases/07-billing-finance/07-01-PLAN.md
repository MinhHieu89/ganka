         - _discounts (List of Discount -- forward-declared, same approach)

         (a) Plan 04 (wave 2) configures EF Core HasMany(Payments/Discounts) with PropertyAccessMode.Field

       - Navigation properties:
         - public IReadOnlyList<InvoiceLineItem> LineItems => _lineItems.AsReadOnly();

       - Properties: InvoiceNumber (string), VisitId (Guid? -- nullable for OTC sales), PatientId (Guid), PatientName (string), Status (InvoiceStatus, default Draft), SubTotal (decimal), DiscountTotal (decimal), TotalAmount (decimal), PaidAmount (decimal), CashierShiftId (Guid?), FinalizedById (Guid?), FinalizedAt (DateTime?), RowVersion (byte[]) for concurrency
       - Computed: BalanceDue => TotalAmount - PaidAmount, IsFullyPaid => BalanceDue <= 0
