**Create clinical-api.ts:**
- Define clinicalKeys query key factory:
  - all: ["clinical"]
  - visits: () => [...all, "visits"]
  - visit: (id: string) => [...visits(), id]
  - activeVisits: () => [...all, "active-visits"]
  - icd10Search: (query: string) => [...all, "icd10", query]
  - doctorFavorites: (doctorId: string) => [...all, "icd10-favorites", doctorId]

- API functions (using openapi-fetch client):
  - getActiveVisits(): GET /api/clinical/active
  - getVisitById(id): GET /api/clinical/{visitId}
  - createVisit(data): POST /api/clinical
  - signOffVisit(visitId): PUT /api/clinical/{visitId}/sign-off
  - amendVisit(visitId, data): POST /api/clinical/{visitId}/amend
  - advanceWorkflowStage(visitId, newStage): PUT /api/clinical/{visitId}/stage
  - updateVisitNotes(visitId, notes): PUT /api/clinical/{visitId}/notes
  - updateRefraction(visitId, data): PUT /api/clinical/{visitId}/refraction
  - addVisitDiagnosis(visitId, data): POST /api/clinical/{visitId}/diagnoses
  - removeVisitDiagnosis(visitId, diagnosisId): DELETE /api/clinical/{visitId}/diagnoses/{diagnosisId}
  - searchIcd10Codes(term): GET /api/clinical/icd10/search?term={term}
  - toggleIcd10Favorite(data): POST /api/clinical/icd10/favorites/toggle
  - getDoctorFavorites(): GET /api/clinical/icd10/favorites

- TanStack Query hooks:
  - useActiveVisits(): useQuery with queryKey=clinicalKeys.activeVisits(), refetchInterval=5000, refetchIntervalInBackground=false
  - useVisitById(id): useQuery with queryKey=clinicalKeys.visit(id)
  - useIcd10Search(term): useQuery with queryKey=clinicalKeys.icd10Search(term), enabled=term.length>=2
  - useDoctorFavorites(): useQuery with queryKey=clinicalKeys.doctorFavorites(currentDoctorId)
  - useCreateVisit(): useMutation with invalidation of activeVisits
  - useAdvanceStage(): useMutation with invalidation of activeVisits
  - useSignOffVisit(): useMutation with invalidation of visit and activeVisits
  - useAmendVisit(): useMutation
  - useUpdateNotes(): useMutation
  - useUpdateRefraction(): useMutation
  - useAddDiagnosis(): useMutation
  - useRemoveDiagnosis(): useMutation
