1. Authenticate first (login to get access token, since endpoints RequireAuthorization)
2. Send GET request with Bearer token
3. Assert response status code is 200 (for valid invoice)
4. Assert Content-Type header contains "application/pdf"
5. Assert response body byte length > 0

Per D-07: response-level validation only, no PDF content parsing.
